# PIZZA
## Presentation: https://github.com/LeonidMehonoshin/PIZZA/blob/main/PresentationUU.pptx
### Installation
Linux/MacOS:
```bash
python -m venv venv
source venv/bin/activate
pip install pygame
python3 main.py
```
Windows:
```powershell
python -m venv venv
venv\Scripts\activate.bat
pip install pygame
python3 main.py
```
